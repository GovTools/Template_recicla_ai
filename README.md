# **Template Recicla AI**

## **Estrutura do Projeto**

```bash
template_recicla_ai/

│   ├── index.html          # Página principal da aplicação
│   ├── input.css           # Arquivo CSS de entrada para o Tailwind

│   └── output.css          # CSS compilado do Tailwind
├── tailwind.config.js      # Configuração do Tailwind CSS
